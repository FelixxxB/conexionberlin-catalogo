## Design Context

### Users
Customers of Conexión Berlín, a board game café in Buenos Aires. Used both in-venue (mobile in hand, picking a game to play) and from home (planning a visit). Primary mobile target is iPhone 11 (375×812px). Equal mix of contexts.

### Brand Personality
Warm, earnest, analog. A neighborhood cultural café — not a tech product. Think dark wood shelves, warm light, books and games everywhere. Intellectual without being pretentious.

3 words: **cálido, cultural, porteño**

### Aesthetic Direction
Dark theme (low-light café environment, evening use). Orange-red brand accent (#e84c1e). Neutrals tinted warm toward the brand hue. Typography pairing: Spectral (serif, display only — for the brand wordmark) + Figtree (warm geometric, all UI text). No Inter.

Anti-references: cold tech dashboards, neon-on-dark gaming UIs, generic admin panels.

### Design Principles
1. **Warmth through restraint** — subtle color tinting, not loud decoration
2. **Rhythm over uniformity** — varied spacing creates hierarchy
3. **Mobile-first reality** — iPhone 11 is primary; every interaction works with one thumb
4. **Data density with breathing room** — 1,532 games needs efficient layout, not sparse whitespace
5. **Compact cards** — user chose compact: image, name, players, time only
